# Codebook

This codebook summarizes the variables retained in the public dataset. Numeric category labels are retained as originally coded and should be interpreted according to the study questionnaire.

## Demographic and relationship variables

`Gender`, `Age_2`, `Marital`, `Parental`, `Education`, `Religion`, `Occupation`, `CoupleStatus`, `Income`, `CouplePeriod`, `NumCouple`, and `NumCouple_2` represent participant demographic and relationship-status variables. Exact age and respondent ID were removed from the public dataset.

## Acceptance of differences items and scores

Item-level variables include `Empathy_*`, `Respect_*`, `Distancing_*`, `Growth_*`, and `Patience_*`. Scale-level variables include `ADCRS`, `TD`, `PD`, `RD`, `GD`, and `DD`.

## Big Five personality variables

Item-level variables include `Extraversion_*`, `Neuroticism_*`, `Openness_*`, `Agreeableness_*`, and `Conscientiousness_*`. Scale-level variables include `Extraversion`, `Neuroticism`, `Openness`, `Agreeableness`, and `Conscientiousness`.

## Communication strategy variables

Item-level variables include `Avoidance_*`, `Dominance_*`, `Compromise_*`, `Accomodation_*`, and `Collaboration_*`. Scale-level variables include `Compromising`, `Integrating`, `Obliging`, `Avoiding`, `Dominating`, `PositiveStrategy`, and `NegativeStrategy`.

## Relationship satisfaction variables

Item-level variables include `satisfaction_*`. The scale-level outcome variable is `CoupleSatisfaction`.
